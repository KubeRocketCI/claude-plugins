# krci-godev

Expert Go developer agent specializing in Kubernetes operators, Custom Resources, and Go best practices.

## Agent

**go-dev** - Automatically activates for Go/Kubernetes operator work

## Commands

- `/krci-godev:review-code <file-path>` - Review Go code against standards (Effective Go, Google Style Guide, operator best practices)
- `/krci-godev:implement-new-cr <cr-name>` - Scaffold and implement Kubernetes Custom Resource with controller

## Skills

- **go-coding-standards** - Idiomatic Go patterns, error handling, testing, security best practices
- **operator-best-practices** - Kubernetes operator development, CRD design, controller patterns, operational practices

## Installation

```bash
/plugin install krci-godev
```
